Clausura de DF

La clausura de un conjunto de dependencias funcionales F, es el conjunto todas las dependencias funcionales inferidas por F.

17

Ejemplo

  • relación:

alumne(legajo, nombre)

  • conjunto de dependencias funcionales:

F = { legajo → nombre }

  • clausura de F:

F ⁺ = { legajo → legajo, nombre → nombre, legajo → nombre, legajo → legajo nombre, legajo nombre → legajo, legajo nombre → nombre, legajo nombre → legajo nombre }

Clausura de conjuntos de atributos

La clausura de un conjunto de atributos X respecto de un conjunto de dependencias F es el conjunto todos los atributos A tal que X → A es inferido por F.
